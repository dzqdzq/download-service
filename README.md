## Usage
1. download docker-compose.yml
2. change user/pass
3. setting download path
3. docker-compose up -d

## Download
```shell
curl -u <curl user>:<curl pass> http://<your ip>:3000/filepath -O -f
```

## UI Admin
1. Open the browser and enter http://\<your ip\>:8080.
2. input admin user/pass
